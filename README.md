# РКН Рулетка (Closed)

[!!!] Этот репозиторий закрыт. Финальная версия вышла, обновлений не будет.

**Сатирическая браузерная игра** о российской интернет-цензуре. Почувствуй себя сотрудником Роскомнадзора и забань ВСЕ!

## О проекте

РКН Рулетка — это пародия блокировок интернет-ресурсов в России. Игра позволяет забанить популярные сервисы по абсурдным причинам, используя различные методы блокировки.

### Особенности

Есть контурент - https://www.rilliat.ru/ban-roulette/, но в нем нет ЭТОГО:

- **Режим безумия (x5!)** - блокируй 3-5 сервисов одновременно!!
- **Добавляй свои сервисы** - расширяй список жертв
- **ДОСТИЖЕНИЯ** - ПОЛУЧАЙ ИХ ВСЕ!

## Установка

Просто открой его в браузере — никаких зависимостей!

## Как играть

1. Нажми **"Заблокировать"** для случайной блокировки одного сервиса!
2. Или активируй **"Режим безумия"** для массовых блокировок (3-5 сервисов)!!!
3. Получай **1₽** за каждую блокировку!
4. Добавляй свои сервисы через кнопку **"+"** в левом нижнем углу!

## Причины блокировки

От **реалистичных**:
- "Распространение вредоносного кода/фишинг"
- "Пропаганда расовой, национальной, религиозной враждебности"

До **САМЫХ абсурдных**:
- "Просто так, хочу :D"
- "Враждебные пиксели"
- "Основатель сервиса не установил MAX"
